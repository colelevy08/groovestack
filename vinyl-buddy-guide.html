Overview

The Vinyl Buddy is an ESP32-based hardware device that sits near your turntable and automatically identifies the record you are playing. It captures a 2-second audio sample via an INMP441 MEMS microphone, sends the raw PCM data to the GrooveStack server over Wi-Fi, and the server queries the AudD audio fingerprinting API to identify the track. The result appears in your GrooveStack listening history within seconds.

The device supports BLE pairing, an SSD1306 OLED display, SD card offline caching, gesture control, mesh networking, MQTT telemetry, capacitive touch inputs, an LED strip for ambient visualization, accelerometer-based motion detection, a real-time clock for accurate timestamps, and a full power-on self-test at startup.

Firmware Flashing

  1. Install the Arduino IDE and add the ESP32 board package (version 2.x+).
  2. Install the required library: ArduinoJson 7.x (via Library Manager). The ESP32 core provides WiFi, I2S, BLE, SD, SPI, Wire, EEPROM, Preferences, and OTA libraries.
  3. Open firmware/vinyl_buddy.ino in the Arduino IDE.
  4. Select ESP32 Dev Module as the board and the correct USB port.
  5. Click Upload. The firmware is a single-file sketch with 64 sections.
  6. On first boot, the device enters AP mode (VinylBuddy-Setup / groovestack) for WiFi configuration via captive portal.
OTA Updates: After initial flash, subsequent firmware updates can be applied over-the-air from the GrooveStack app. The firmware also supports automatic rollback if a new version fails to boot.

Device Setup & Pairing

  1. Flash the firmware from the firmware/ directory to the ESP32.
  2. On first boot, the device starts in AP mode. Connect to the VinylBuddy-Setup WiFi network (password: groovestack) and enter your home WiFi credentials via the captive portal.
  3. The device reboots, connects to WiFi, and displays a 12-character device code on the OLED screen.
  4. In GrooveStack, navigate to the Vinyl Buddy tab and enter the device code to pair.
  5. The device sends heartbeats every 30 seconds to confirm connectivity.
  6. Place the device near your turntable speaker and start playing a record. The POST routine validates all hardware at startup.

Troubleshooting

Device not pairing

  • Ensure the device is powered on and displaying a code on the OLED screen.
  • Verify the device is connected to a WiFi network with internet access.
  • Check that the backend URL in the firmware (API_BASE_URL) matches your GrooveStack server.
  • Try power-cycling the device and waiting 30 seconds for the code to regenerate.

Audio not identifying

  • Run the calibration routine from the app to adjust for your room acoustics.
  • Move the device closer to the turntable speaker.
  • Check the Audio Quality indicator in the web dashboard — minimum RMS of 200 is required.
  • Verify your AUDD_API_TOKEN is valid and has remaining credits.

Firmware update fails

  • Ensure stable WiFi connection during the update process.
  • Use the /api/vinyl-buddy/firmware/versions endpoint to verify the target version exists.
  • If a bad firmware is flashed, the automatic rollback (Section 63) will revert to the previous version after 3 consecutive boot failures.
  • As a last resort, reflash via USB using the Arduino IDE.

I2C devices not detected

  • Use the I2C Bus Scanner (HW-15) in the web interface to check for detected addresses.
  • Verify wiring: SDA to GPIO 21, SCL to GPIO 22, with appropriate pull-up resistors.
  • Check for I2C address conflicts (MPU6050 and DS3231 both use 0x68 by default).
